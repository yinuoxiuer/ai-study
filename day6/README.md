# Python 面向对象编程与异常处理 (Day 6)

本项目包含 Day 6 学习过程中的所有 Python 脚本，主要涵盖了面向对象编程（OOP）的核心概念、设计模式以及异常处理机制。

## 文件说明

### 面向对象编程 (OOP) 核心

*   **封装.py**
    *   演示类的基本定义、属性封装以及 `__init__`、`__str__` 等内置方法的使用。
    *   案例：小明爱跑步（体重增减）。

*   **继承.py**
    *   演示单继承、方法重写（Override）以及使用 `super()` 调用父类方法。
    *   案例：动物 -> 狗 -> 哮天犬。

*   **多态.py**
    *   演示多态的实现，即子类重写父类方法，在调用时表现出不同的行为。
    *   案例：人与不同品种的狗玩耍。

*   **多重继承.py** & **多继承的init.py**
    *   演示一个子类继承多个父类的情况。
    *   包含 `__mro__`（方法解析顺序）的查看以及在多继承环境下如何正确使用 `super().__init__` 传递参数。

*   **私有属性和私有方法.py**
    *   演示如何使用双下划线 `__` 定义私有成员，以及 Python 中“伪私有”的实现原理（名称重整）。

*   **类属性.py**
    *   演示类属性（静态变量）、类方法（`@classmethod`）和静态方法（`@staticmethod`）的区别与应用。

### 面向对象实战案例

*   **摆放家具.py**
    *   演示对象之间的交互与依赖关系。
    *   案例：房子类添加家具对象，计算剩余面积。

*   **士兵与枪.py**
    *   演示复杂的对象关联。
    *   案例：士兵使用枪类对象进行射击。

### 设计模式

*   **单例模式.py**
    *   演示如何通过重写 `__new__` 方法实现单例模式，确保一个类只有一个实例，并控制初始化动作只执行一次。

### 异常处理

*   **简单的异常捕获.py**
    *   演示 `try...except...else...finally` 的完整语法。
    *   包含捕获特定异常、捕获未知异常以及获取异常详细信息（报错文件和行号）。

*   **异常的传递.py**
    *   演示异常在函数调用栈中的传递机制，只需在主程序中捕获即可。

*   **主动抛出异常.py**
    *   演示使用 `raise` 关键字主动抛出异常，以及 `assert` 断言的使用。

*   **test_exception_module.py**
    *   用于演示跨模块异常捕获的辅助模块。

### 其他

*   **使用内置方法.py**
    *   演示 `__del__`、`__str__` 等内置方法的触发时机。

*   **循环读取.py**
    *   简单的输入处理示例。

## 核心知识点总结

1.  **封装**：将属性和方法封装到一个抽象的类中。
2.  **继承**：实现代码复用，子类拥有父类的所有属性和方法。
3.  **多态**：不同的子类对象调用相同的父类方法，产生不同的执行结果。
4.  **单例**：确保全局只有一个实例对象。
5.  **异常处理**：增强程序的健壮性，通过 `try-except` 捕获并处理运行时错误。
