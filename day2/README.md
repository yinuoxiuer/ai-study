# Day 2 Python 学习笔记

本项目包含了 Day 2 学习过程中编写的所有 Python 脚本，涵盖了基础语法、变量、数据类型、运算符以及一些练习题。

## 文件列表及简介

### 基础概念与语法
*   **main.py**: 包含一个简单的 `myfunc` 函数，打印 "hello world"，演示了基本的函数定义和调用。
*   **debug.py**: 演示了简单的打印语句和变量赋值，用于调试练习。
*   **annote.py**: 演示了 Python 中的注释写法以及基本的代码执行流程。
*   **output.py**: 演示了格式化输出，包括字符串、整数和浮点数的格式化 (`%s`, `%d`, `%.2f`)。
*   **datatype.py**: 演示了 `input()` 函数的使用以及数据类型转换（将字符串转换为浮点数）。
*   **variable.py**: 演示了浮点数变量的定义和格式化输出。
*   **arithmetic.py**: 演示了各种算术运算符（除法）、比较运算符、逻辑运算符（`and`, `or`）以及位运算符（`&`, `|`, `^`, `~`, `<<`, `>>`）。

### 练习题 (Homework)
*   **homeworlk1.py**: 打印不同数据类型的类型信息 (`type()`)，包括字符串、整数、浮点数、布尔值、None、列表等。
*   **homework1.py**: (空文件) 预留的作业文件。
*   **homework2.py**: 演示了浮点数格式化输出和整数转字符串。
*   **homework3.py**: 使用 `for` 循环打印 0 到 49 之间的奇数 (2*i+1)。
*   **homework4.py**: 打印九九乘法表（部分实现），演示了嵌套循环和 `print` 函数的 `end` 参数。
*   **homework5.py**: 接收用户输入的整数，打印其二进制表示，并计算二进制中 1 的个数（通过位移和取模实现）。

## 运行说明

可以直接使用 Python 解释器运行上述任意文件，例如：

```bash
python main.py
python homework5.py
```

请确保已安装 Python 环境。
