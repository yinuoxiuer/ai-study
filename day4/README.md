# Python 学习笔记与示例

本项目包含一系列 Python 脚本，用于演示 Python 的基础语法、数据结构及常用操作。主要涵盖了元组、字典、集合、字符串的操作，以及内存管理、函数参数与返回值等概念。

## 文件说明

### 数据结构操作

*   **元组.py**
    *   演示元组的定义与基本使用。
    *   包含元组格式化输出、单元素元组的定义 `(1,)` 以及元组的遍历。

*   **字典.py**
    *   展示字典的创建、修改、查找及删除操作。
    *   包含 `update`、`pop`、`setdefault` 方法的使用。
    *   演示字典的遍历（键、值、键值对）。

*   **集合.py**
    *   演示集合（Set）的创建与基本运算。
    *   包含添加元素 `add`、差集 `difference`、对称差集 `symmetric_difference`。
    *   包含集合的拷贝、判断是否不相交 `isdisjoint` 以及集合推导式。

*   **字符串.py**
    *   演示字符串的常用方法，如 `upper`、`find`、`strip`、`split`、`join`。
    *   包含转义字符的使用示例。

*   **列表切片.py**
    *   演示列表的切片赋值操作，例如 `test_list[3:3]=["x","y","z"]`。

*   **字符串切片.py**
    *   演示字符串的切片操作，包括步长、反转字符串。
    *   展示字符串与列表之间的相互转换。

*   **容器的公共方法.py**
    *   演示容器类型的通用操作。
    *   包含 `zip` 函数用于打包数据，`enumerate` 用于枚举。
    *   展示列表推导式与字典推导式的应用。

*   **元组生成式.py**
    *   演示如何使用生成器表达式创建元组。

### 函数与内存

*   **函数返回值.py**
    *   演示函数返回多个值（本质是返回元组）。
    *   展示变量解包（Unpacking）及交换变量值的操作。

*   **缺省参数.py**
    *   演示函数定义中的默认参数（缺省参数）的使用规则。

*   **内存管理.py**
    *   探讨 Python 的内存管理机制。
    *   演示 `id()` 函数、`is` 运算符的使用。
    *   展示小整数对象池、字符串驻留机制以及对象的引用与删除。

### 其他

*   **main.py**
    *   PyCharm 默认生成的入口脚本示例。

## 运行方式

可以直接运行各个 `.py` 文件来查看对应的输出结果和演示效果。
例如：
```bash
python 字典.py
```
