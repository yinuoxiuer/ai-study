# 第15天：机器学习模型

该项目包含几种常见的机器学习模型的实现。

## 文件

*   `main.py`: 项目的主入口。
*   `Decision_Tree.ipynb`: 包含决策树模型的实现和可视化的Jupyter Notebook。
*   `knn_learn.ipynb`: 包含K-近邻（k-NN）算法实现的Jupyter Notebook。
*   `bayess_learn.ipynb`: 包含朴素贝叶斯分类器实现的Jupyter Notebook。
*   `tree.dot`, `tree.png`, `tree1.dot`: 决策树模型的可视化文件。

---

*数据来源：本项目使用的数据来自第14天的`data`文件夹。*
