# Day 7: Python 模块与包学习

本项目包含了 Python 模块、包以及面向对象编程中关于私有属性继承的学习示例。

## 项目结构

- `module1.py`, `module2.py`: 模块定义示例。
- `导入重名.py`: 演示如何处理模块中重名函数的导入冲突。
- `使用包.py`: 演示如何导入和使用 Python 包。
- `wd_message/`: 一个自定义包的示例目录。
- `子类使用父类的私有属性.py`: 演示 Python 中子类对父类私有属性的访问限制及处理方式。
- `new.sh`: 可能用于创建新文件或初始化环境的脚本。

## 学习重点

1. **模块 (Modules)**: 如何定义和导入模块。
2. **包 (Packages)**: 如何组织多个模块。
3. **命名冲突**: 使用 `as` 关键字或完整路径解决重名问题。
4. **继承与私有属性**: 理解 Python 中的封装性，以及子类无法直接访问父类私有属性的特性。
