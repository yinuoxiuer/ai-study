# Day 3 Python 学习笔记

本项目包含 Python 基础语法的学习代码，主要涵盖循环、函数、列表操作、变量作用域及模块使用等内容。

## 文件说明

### 1. 循环与控制流
- **循环.py**: 演示 `while` 循环和 `for` 循环的基本用法。
- **break用法.py**: 演示在 `while` 循环中使用 `break` 语句提前退出循环。

### 2. 函数与模块
- **函数参数.py**: 定义简单的函数，演示参数传递。
- **使用模块.py**: 演示如何导入自定义模块并调用其中的函数。
- **second_module.py**: 定义工具函数，并使用 `if __name__ == "__main__":` 进行模块测试。
- **全局变量的模块测试.py**: 测试导入模块时的变量作用域问题。
- **random learn.py**: 演示 `random` 模块生成随机数。

### 3. 列表操作
- **列表初步.py**: 列表的创建、修改、追加 (`append`)、插入 (`insert`)、清空 (`clear`) 及删除 (`del`)。
- **列表简写.py**: 列表的拼接 (`+`)、重复 (`*`)、扩展 (`extend`) 及切片操作。
- **列表数据统计.py**: 列表长度计算 (`len`)、排序 (`sort`) 及遍历。
- **列表生成式.py**: 演示列表推导式（List Comprehensions）的高级用法，包括嵌套循环和条件判断。
- **作业答案4.py**: 使用列表生成式生成 100 以内的奇数列表。

### 4. 变量作用域与类型
- **局部变量与全局变量.py**: 演示函数内部访问全局变量。
- **变量重名.py**: 演示局部变量与全局变量重名时的“就近原则”，以及 `global` 关键字的使用。
- **可变与不可变类型变量.py**: 通过打印内存地址 (`id()`)，演示可变类型（如列表）与不可变类型（如整数）在函数传参和赋值时的区别。

## 运行说明
可以直接运行各个 `.py` 文件来查看对应的输出结果。例如：
```bash
python 循环.py
```
