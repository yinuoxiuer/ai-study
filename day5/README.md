# Day 5 Python 学习笔记

本项目包含 Day 5 的 Python 学习代码，主要涵盖了函数参数、递归、面向对象基础以及一些常用的 Python 技巧。

## 文件说明

### 1. [判空.py](判空.py)
- 演示了 Python 中空容器（如列表）的布尔值判定。
- 说明了空容器默认为 `False`，以及与 `None` 的区别。

### 2. [递归.py](递归.py)
- 包含递归函数的示例：
  - `print_info(num)`: 递归打印数字。
  - `sum_num(num)`: 递归计算累加和。
  - `step(num)`: 解决台阶问题的递归实现（类似斐波那契数列）。
- 设置了递归深度限制 `sys.setrecursionlimit`。

### 3. [多值参数.py](多值参数.py)
- 演示了可变参数的使用：
  - `*args`: 接收位置参数元组。
  - `**kwargs`: 接收关键字参数字典。
- 展示了参数解包（Unpacking）的用法。

### 4. [缺省参数.py](缺省参数.py)
- 演示了函数定义中的默认参数值（缺省参数）。
- 展示了如何在调用时覆盖默认值或使用关键字参数指定特定参数。

### 5. [面向对象.py](面向对象.py)
- 介绍了 Python 面向对象编程（OOP）的基础。
- 定义了 `person` 和 `animal` 类。
- 演示了 `__init__` 构造方法、实例属性（`self.name` 等）和实例方法。
- 展示了对象的创建、属性访问及修改。

### 6. [main.py](main.py)
- PyCharm 生成的默认示例脚本，包含简单的函数调用。
